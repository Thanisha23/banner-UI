## Project Setup

1. **Clone the repository**
```
git clone https://github.com/Thanisha23/banner-UI.git
```

2. **Navigate to the project directory and install dependencies for the backend**

 ```
pnpm install
```

3. **Start the server**
```
pnpm run dev
```
